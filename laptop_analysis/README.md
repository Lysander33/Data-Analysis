# 印度笔记本电脑规格与价格数据分析

数据来源 [Kaggle Laptop Price Dataset](https://www.kaggle.com/datasets/ionaskel/laptop-prices)，涵盖印度市场 1,274 款笔记本的规格和价格信息。价格单位为印度卢比 (₹, INR)，1 USD ≈ 83 INR (2024)。

## 快速开始

```bash
pip install -r requirements.txt
python data_analysis.py
```

脚本默认自动扫描 `outputs/` 下最新的 `my_cleaned_data.jsonl`，也可手动指定输入：

```bash
python data_analysis.py --input outputs/<job_id>/my_cleaned_data.jsonl
```

## 核心发现

### 价格分布

价格呈明显右偏分布 —— 多数机型集中在中低价位，少量高端机型拉高均值。

| 价格区间 | 说明 |
|----------|------|
| ₹9,271 — ₹15,000 | 入门级（Chuwi、Vero、Mediacom 等低端品牌） |
| ₹15,000 — ₹32,496 (Q1) | 中低价位，覆盖多数 Notebook |
| ₹32,496 — ₹52,694 (中位数) | 主流价位段 |
| ₹52,694 — ₹79,773 (Q3) | 中高价位，Ultrabook 和入门游戏本 |
| ₹79,773 — ₹150,690 | 高端机型（Apple、Razer、MSI） |
| > ₹150,690 | 离群区（工作站、顶配游戏本） |

### CPU 市场格局

| CPU 品牌 | 数量 | 占比 | 均价 (₹) |
|----------|------|------|-----------|
| Intel | 1,213 | 95.2% | 62,038 |
| AMD | 60 | 4.7% | 29,890 |
| 其他 | 1 | 0.1% | 35,112 |

**Intel 绝对主导**，AMD 仅在低价位机型中占有一席之地。Intel 机型均价约为 AMD 的 2.1 倍，但这主要反映 AMD 集中于低端市场而非技术劣势。

### GPU 分析

| GPU 品牌 | 数量 | 均价 (₹) | 类型 |
|----------|------|-----------|------|
| Intel | 703 | 54,415 | 集成显卡 |
| Nvidia | 396 | 79,747 | 独立显卡 |
| AMD | 174 | 41,453 | 集成+少量独显 |
| ARM | 1 | 35,112 | 集成 |

- **独立显卡溢价显著**：搭载 Nvidia 独显的机型均价比 Intel 集显机型高 46.6%（Welch's t-test, p < 0.001）
- **集显占主流**：55.2% 的机型使用 Intel 集成显卡

| Windows 7 | 45 | 89,865 |

macOS 机型（Apple）均价最高；Windows 10 覆盖 82.2% 的机型；无系统 / Linux / Chrome OS 集中在低价位。

### 内存与存储

**内存是价格的最强单一预测因子**（Pearson r = 0.62，与价格的线性相关性在各项硬件指标中最高）。

**存储类型价格差异：**

SSD 机型比 HDD 机型平均贵 2.3 倍（Welch's t-test, p < 0.001），差异极其显著。SSD 已成为中高端机型标配。

### 屏幕尺寸与价格

屏幕尺寸和价格之间存在非线性关系 —— 小屏和大屏都倾向于高价，中等尺寸（14-15.6 英寸）覆盖主流价位段。

### 重量与价格

| 重量段 | 定位 |
|--------|------|
| 轻薄 (<1.5kg) | 高价 Ultrabook，便携溢价 |
| 中等 (1.5-2.0kg) | 主流全能本 |
| 偏重 (2.0-2.5kg) | 中端游戏本和商务本 |
| 厚重 (>2.5kg) | 高端游戏本和工作站，性能溢价 |

重量与价格呈 U 型关系：最轻和最重的机型都贵，前者为工程溢价，后者为性能溢价。

### 特征相关性矩阵

| Pearson r | 价格 | 英寸 | 内存 | 重量 |
|-----------|------|------|------|------|
| **价格** | 1.00 | — | — | — |
| **英寸** | ~0.18 | 1.00 | — | — |
| **内存(GB)** | **0.62** ★ | ~0.15 | 1.00 | — |
| **重量(kg)** | ~0.32 · | ~0.55 | ~0.35 | 1.00 |

★ 强相关 (|r|>0.5)　· 中等相关 (|r|>0.3)

**解读**：内存与价格强正相关（每增加 1GB 内存，价格上升约 ₹2,600）；屏幕尺寸与价格仅有弱相关，说明大屏并不等于高价。

### 线性回归：价格 ~ 内存 + 英寸 + 重量

| 变量 | 系数 | 含义 |
|------|------|------|
| 截距 | ≈ -₹40,000 | 基准调整项 |
| 内存 | ≈ +₹2,600 / GB | 每增加 1GB 内存的价格增量 |
| 英寸 | ≈ +₹2,000 / inch | 每增加 1 英寸屏幕的价格增量 |
| 重量 | ≈ +₹6,000 / kg | 每增加 1kg 重量的价格增量 |

多元 OLS 回归，R² ≈ 0.42。内存是解释力最强的变量。

## 分析内容

### 终端输出（19 项统计）

价格统计、品牌/类型/CPU/GPU 对比、内存与存储分析、屏幕与重量趋势、相关性矩阵、OLS 线性回归、Welch's t-test 假设检验、IQR 离群值检测、品牌×类型交叉统计。

### 可视化图表（15 张，运行脚本生成）

直方图、条形图、箱线图、饼图、散点图、热力图 — 覆盖价格分布、品牌对比、硬件配置与价格关系、相关性分析等维度。图表输出至 `outputs/charts/`。

## 数据清洗流水线

`my_config.yaml` 定义清洗步骤（依赖 `mini-dataset-processing` 框架）：

1. **文本长度过滤** — 丢弃过短/过长记录
2. **链接清洗** — 移除文本中的 URL
3. **文档去重** — 移除重复记录

清洗产出流向 `outputs/<job_id>/my_cleaned_data.jsonl`。

## 技术说明

- 统计方法：Pearson 相关系数、Welch's t-test（正态近似）、IQR 离群值检测、OLS 线性回归 — 纯 Python + numpy 实现，不依赖 scipy
- 编码处理：Win32 平台自动重配 stdout/stderr 为 UTF-8，确保 ₹ 符号正常输出
- 图表字体：运行时自动探测系统可用的中文字体
